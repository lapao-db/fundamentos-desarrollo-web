¿Qués es HTML?

Lenguaje de marcado que estructura el contenido de una página web. Basado en etiquetas que pueden ser semánticas o no semánticas.

Recuerda que ¡No es un lenguaje de programación!

Etiquetas Básicas de HTML

Etiqueta Descripción
h1, h2, h3, h4, h5, h6 Titulos y subtitulos en HTML.
p parrafos
img muestra imagenes (atributos src y alt)
ul, ol Definir listas desorganizadas y organizadas.

Formularios en HTML

Herramienta fundamental para recopilar datos de un usuario, a traves de diferentes campos como:

  • Textos. Para pedir nombre, contraseña
  • Casillas de texto - áreas de texto. Descripciones mas largas, pregunta abierta.
  • Listas desplegables - rango de edad.
  • Botones
  • Subir archivos - file
  • Checkbox
  • Fecha - date
  • Rangos - range
  • Buscar - search
